#Party Time - 会聚
- - -
<font size=3>该应用设计的初衷是为了解决大学生聚会选址问题。主要功能包括基于位置的活动检索、基于多个位置离散用户选择最佳聚会位置、发送活动邀请、聚会投票等功能。</font>
<br/><br/>
